This contains quick descriptions of what each of these folders contain

Code Descriptions: all files in this folder are the same thing: icd10 code-to-description mappings.

Updated Codes from previous years: every year, the codes denoting a condition can slightly change. The files here contain all the updated codes, useful for updating ICD10 systems.

Table and Index - contains the alphabetical index (disease name -> general code) and tabular list (general code -> disease name and more specific codes relating to that disease)
